## 如何运行原型

### 方法1：直接用 Godot 打开

1. 确保已安装 Godot 4.6+
2. 克隆仓库后，用 Godot 打开项目文件夹
3. 点击运行（F5）

### 方法2：CLI 运行

```bash
cd ~/goblin-rhythm-prototype
godot --headless --script test_prototype.gd
```

### 验证清单

原型应该包含以下功能：

- [ ] 按空格开始游戏
- [ ] 每120BPM（每秒2拍）触发节拍
- [ ] 节拍时屏幕底部的绿色条会闪一下
- [ ] 每4拍切换一次预期指令（↑ ↓ ← →）
- [ ] 按 WASD 或方向键进行节奏输入
- [ ] 正确输入：机器执行动作，建筑受到伤害
- [ ] 错误输入：机器卡壳（抖动效果）
- [ ] 连击系统：连续正确输入增加combo
- [ ] 破坏度达到100%时显示胜利

### 测试按键

| 按键 | 动作 |
|------|------|
| W / ↑ | 向上 |
| S / ↓ | 向下 |
| A / ← | 向左 |
| D / → | 向右 |
| Space | 开始游戏 |
| Escape | 暂停 |
